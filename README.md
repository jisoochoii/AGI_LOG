## 프로젝트 소개 💸

> **_아기의 건강을 관리하자_**

기존 서비스의 육아일기 말고 매일 아이의 건강을 기록하여, 아이가 아플 때 전문의에게 진료를 더 편히 볼 수 있게 해주는 웹 서비스 입니다.
